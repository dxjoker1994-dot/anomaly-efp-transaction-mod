# Anomaly EFP Transaction Mod

A standalone mod for S.T.A.L.K.E.R. Anomaly that adds comprehensive transaction logging and financial tracking to the PDA Guide section.

## Features

✅ **Transaction Logging**
- Records all income and expenses in real-time
- Persistent transaction history
- Detailed transaction information (amount, category, timestamp)

✅ **Category Tracking**
- Weapons
- Medical Supplies
- Repairs & Maintenance
- Supplies & Consumables
- Miscellaneous
- Quest Rewards

✅ **PDA Integration**
- New "Finance" section in PDA Guide
- View all transactions
- Filter by category
- Summary statistics (total income, total expenses, net balance)

✅ **Quest Completion**
- Automatic income logging when quests are completed
- Quest reward tracking
- Quest history in transaction log

## Installation

1. Download the mod folder
2. Place `anomaly-efp-transaction-mod` in your `Anomaly/mods/` directory
3. Launch the game - mod will auto-load

## Usage

1. Open your PDA (default: Tab key)
2. Navigate to Guide section
3. Select "Finance" to view transactions
4. Browse transaction history filtered by category
5. Check summary statistics for account overview

## Configuration

Edit `gamedata/configs/transaction_log.ltx` to customize:
- Transaction categories
- Display formats
- Log retention settings

## Compatibility

- S.T.A.L.K.E.R. Anomaly 1.5+
- Compatible with other mods (non-intrusive)
- EFP mod framework compatible
